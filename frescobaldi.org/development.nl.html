De ontwikkeling van Frescobaldi

Aan Frescobaldi 2.0 wordt op dit moment gewerkt in git op Github. Frescobaldi 2 is geschreven in Python, met PyQt4 voor de gebruikersinterface, en is ontworpen om op alle belangrijke platformen (Linux, Mac OS X en MS Windows) te kunnen worden gebruikt. Frescobaldi 3 gebruikt PyQt5.

Frescobaldi 1.2.x wordt ontwikkeld in bug-fix mode in Googlecode's Subversion depot. Deze versie gebruikt de KDE4-bibliotheken en is daardoor alleen beschikbaar op platformen waar de Python-KDE4 bindings beschikbaar zijn.

Frescobaldi is Vrije Software onder de GNU General Public License (GPL). Iedereen is welkom om fouten te vinden en mogelijkheden of vertalingen toe te voegen.

Fouten en vertalingen

Als u een fout ontdekt kunt u deze rapporteren in onze issue tracker (als de fout niet al bekend is). Als u Frescobaldi wilt vertalen in uw eigen taal, lees dan het document README-translations.

Gewenste mogelijkheden

Als u graag bepaalde functionaliteit toegevoegd zou zien kunt u uw wensen kenbaar maken in de issue tracker.

Broncode

Als u de broncode wilt uitchecken om mee te helpen aan de ontwikkeling van Frescobaldi, dan is het aanbevolen om de meest recente broncode te gebruiken uit het GitHub depot. U kunt de broncode verkrijgen door Git op uw computer te installeren en dan de volgende commando's in te geven.

  • Frescobaldi 2 en 3 worden in Git ontwikkeld op github.com/frescobaldi/frescobaldi:
    • git clone git://github.com/frescobaldi/frescobaldi.git
cd frescobaldi
  • Om de 1.x branch (voor KDE4) te krijgen:
    • svn checkout http://lilykde.googlecode.com/svn/trunk/frescobaldi
cd frescobaldi

Het bestand README-development bevat meer informatie over de opzet van de broncode. Als u uw wijzigingen aan Frescobaldi wilt toevoegen, stuur dan een patch naar de mailinglist of het emailadres van de auteur. Gebruik het volgende commando om een patch te maken met uw wijzigingen t.o.v. een recente Git-versie: 

git diff > mypatch.diff

U kunt ook git format-patch gebruiken om commits via e-mail te versturen. Het is niet noodzakelijk om bijgewerkte vertalingen als patches te versturen; stuur deze a.u.b. als complete bestanden.

Zie github.com/frescobaldi/frescobaldi voor meer informatie.
