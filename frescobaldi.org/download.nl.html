Frescobaldi downloaden

Mac OS X

Op Mac OS X is de aanbevolen installatiemethode via MacPorts. Hier zijn instructies (English, français, italiano) van Davide Liessi, die de MacPorts-versie van Frescobaldi bijhoudt.

Installeer ook een exemplaar van LilyPond.

Windows

Voor MS Windows is er een complete installer die Frescobaldi en alle benodigde modules installeert (het "Frescobaldi Setup.exe" bestand). Het kan nodig zijn de Visual C++ 2008 runtime module van Microsoft te installeren.

En installeer natuurlijk een versie van LilyPond!

Linux

De meeste Linux-distributies hebben Frescobaldi in het pakketbeheer opgenomen. Op de Debian-gebaseerde distributies zoals Ubuntu volstaat een eenvoudig commando sudo apt-get install frescobaldi.

Mocht Frescobaldi niet voorhanden zijn dan kan het vrij eenvoudig van broncode geïnstalleerd worden, zie onder.

Van broncode installeren

U kunt Frescobaldi 2 downloaden van de GitHub project releases pagina.

Voor het installeren van broncode is er een installeerbare Python-module (het "frescobaldi-2.x.x.tar.gz" bestand). Deze kan op alle besturingssystemen worden gebruikt. Pak hem uit en lees het INSTALL bestand voor verdere informatie.

Frescobaldi heeft de Python-interpreter versie 2.6 of 2.7 nodig en de PyQt4-bibliotheek. Optioneel, maar sterk aanbevolen, zijn Python-Poppler-Qt4 voor de PDF-weergave, en Python-PortMidi voor de MIDI-weergave.

Installatievolgorde:

  • Installeer Python 2.6 of 2.7 (niet 2.7.0)
  • Installeer PyQt4, voor dezelfde Python-versie uiteraard.
  • Installeer Frescobaldi (werkt dan, maar nog zonder PDF en MIDI-ondersteuning)
  • Installeer Python-Poppler-Qt
  • Installeer Python-PortMidi. Mocht deze lastig te installeren zijn, dan kan ook Pygame worden geïnstalleerd, deze bevat ook de Python-PortMidi bibliotheek.

Voor Linux kunnen alle afhankelijkheden via de distributie of handmatig worden geïnstalleerd.

Op de Linkspagina kunt u verwijzingen vinden naar installeerbare pakketten van Frescobaldi voor specifieke besturingssystemen.

Frescobaldi 1:

U kunt de broncode voor Frescobaldi 1.x (de oude stabiele versie voor KDE4) van de GoogleCode project pagina's downloaden. Frescobaldi 1 werkt alleen op platformen waar KDE4 beschikbaar is. U hoeft niet de KDE Desktop Environment te gebruiken om Frescobaldi te kunnen starten; het is voldoende als de basisbibliotheken van KDE geïnstalleerd zijn.
