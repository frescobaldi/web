Frescobaldi downloaden

Mac OS X

Voor Mac OS X zijn DMG disk images beschikbaar die een applicatiebundel bevatten die je in je Applicaties folder kunt slepen. 64 bit en 32 bit versies zijn beschikbaar (de "Frescobaldi-2.x.x-x86_64.dmg" en "Frescobaldi-2.x.x-i386.dmg" bestanden, respectievelijk). Daarnaast is het mogelijk om op Mac OS X Frescobaldi via Homebrew of MacPorts te installeren. Hier zijn instructies: English, français, italiano.

Welke installatiemethode u ook gebruikt, het is handig om ook te lezen hoe u de MIDI-weergave in Frescobaldi inschakelt (français, italiano).

Installeer ook een exemplaar van LilyPond.

Windows

Voor MS Windows is er een complete installer die Frescobaldi en alle benodigde modules installeert (het "Frescobaldi.Setup.x.y.z.exe" bestand). Het kan nodig zijn de Visual C++ 2008 runtime module van Microsoft te installeren.

Het wordt aanbevolen bij het installeren van een nieuwe versie van Frescobaldi eerst de oude versie te de-installeren.

En installeer natuurlijk een versie van LilyPond!

Linux

De meeste Linux-distributies hebben Frescobaldi in het pakketbeheer opgenomen. Op de Debian-gebaseerde distributies zoals Ubuntu volstaat een eenvoudig commando sudo apt-get install frescobaldi.

Mocht Frescobaldi niet voorhanden zijn dan kan het vrij eenvoudig van broncode geïnstalleerd worden, zie onder.

Van broncode installeren

U kunt Frescobaldi downloaden van de GitHub project releases pagina.

Voor het installeren van broncode is er een installeerbare Python-module (het "frescobaldi-x.y.z.tar.gz" bestand). Deze kan op alle besturingssystemen worden gebruikt. Pak hem uit en lees het INSTALL bestand voor verdere informatie.

Frescobaldi 2 heeft de Python-interpreter versie 2.7+ nodig (niet 2.7.0) of hoger (Python 3.2+ werkt ook!), de PyQt4-bibliotheek en het python-ly pakket. Optioneel, maar sterk aanbevolen, zijn Python-Poppler-Qt4 voor de PDF-weergave, en Python-PortMidi voor de MIDI-weergave.

Installatievolgorde:

  • Installeer Python 2.7+ (niet 2.7.0)
  • Installeer PyQt4, voor dezelfde Python-versie uiteraard.
  • Installeer python-ly
  • Installeer Frescobaldi (werkt dan, maar nog zonder PDF en MIDI-ondersteuning)
  • Installeer Python-Poppler-Qt
  • Installeer Python-PortMidi. Mocht deze lastig te installeren zijn, dan kan ook Pygame worden geïnstalleerd, deze bevat ook de Python-PortMidi bibliotheek.

Frescobaldi 3 Python 3.4 of hoger nodig, PyQt5 en Python-Poppler-Qt5. En uiteraard ook python-ly en Python-PortMidi, geïnstalleerd met Python3.

Installatievolgorde:

  • Installeer Python 3.6 of hoger
  • Installeer PyQt5, voor dezelfde Python-versie uiteraard.
  • Installeer python-ly
  • Installeer Frescobaldi (werkt dan, maar nog zonder PDF en MIDI-ondersteuning)
  • Installeer Python-Poppler-Qt5
  • Installeer qpageview (voor Frescobaldi >= 3.2)
  • Installeer Python-PortMidi. Mocht deze lastig te installeren zijn, dan kan ook Pygame worden geïnstalleerd, deze bevat ook de Python-PortMidi bibliotheek. Gebruik de goede Python versie!

Voor Linux kunnen alle afhankelijkheden via de distributie of handmatig worden geïnstalleerd.

Op de Linkspagina kunt u verwijzingen vinden naar installeerbare pakketten van Frescobaldi voor specifieke besturingssystemen.

Frescobaldi 1:

U kunt de broncode voor Frescobaldi 1.x (de oude stabiele versie voor KDE4) van de GoogleCode project pagina's downloaden. Frescobaldi 1 werkt alleen op platformen waar KDE4 beschikbaar is. U hoeft niet de KDE Desktop Environment te gebruiken om Frescobaldi te kunnen starten; het is voldoende als de basisbibliotheken van KDE geïnstalleerd zijn.
